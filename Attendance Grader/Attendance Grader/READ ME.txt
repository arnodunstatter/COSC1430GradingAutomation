In order to use this:
1. Make sure you have the roster.txt in the appropriate place.
2. Download the attendance files from teams. Save as "attendance" with ANSI encoding. The default UTF encoding will give weird symbols instead of plain text.
3. Download the appropriate evalutaion documents from zybooks. Make sure to not include the information about time-taken. Save as "problems" with ANSI encoding. The default UTF encoding will give weird symbols instead of plain text.
